# findings/novel/

Git-tracked library of **novel** attack payloads -- techniques not previously
documented in the seed library or existing taxonomy.

**Storage threshold**: `novelty_score >= 7` (regardless of vulnerability score)

A novel finding that scores 0 on vulnerability is still recorded here -- it
represents a new attack principle that may need a different target or iteration
to become effective.

When `novelty_score >= 7`, also create a disclosure report in `disclosures/`.
